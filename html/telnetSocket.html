Telnet Socket Module¶

class modules.telnetSocket.telnetSocket(msg='Going well')[source]¶

Establish a socket connection usable through telnet to monitor a process by relaying messages from the process. when the user makes a telnet (or other) socket connection, the class will relay the message to the client. Follow Instructions to get started.

estConn()[source]¶

method to establish the socket connection and receive data from telnet

relayMessages(*msg)[source]¶

relay the messages to the socket if a connection is made

startSocketThread()[source]¶

run socket through a thread in order to allow parallel operation

import telnetSocket
m1= telnetSocket.telnetSocket()
m1.startSocketThread()
m1.t1.start()

###now nc localhost 50007
##or telnet localhost 50007

m1.relayMessages()  ##will relay messages to telnet as long as connected
##or
m1.conn.send("hi")  ##send message to socket even if not connected

##after close execute these commands to let the connection close out.
m1.keepConnection=False
m1.estConnection =False
m1.t1.join()
